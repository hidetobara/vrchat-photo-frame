This is the top page for Web Photo Frame. We supply these API.
  • /sheet/[key]/[worksheet] : the infomation of this spread sheet
  • /img/[key]/[worksheet]/[name] : the named image, deprecated
  • /delete/[key]/[worksheet]/ : delete these images
  • /delete/[key]/[worksheet]/[name] : delete this image
  • /upload/[key]/[worksheet]/[name] : upload this image to CDN
